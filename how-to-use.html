How to Use the Text Converter Tool

  1. ऊपर दिए गए टेक्स्ट बॉक्स में अपना टेक्स्ट टाइप करें या पेस्ट करें।
  2. जिस फॉर्मेट में बदलना चाहते हैं, उस बटन पर क्लिक करें:
    • UPPERCASE: सभी अक्षर बड़े में बदलें
    • lowercase: सभी अक्षर छोटे में बदलें
    • Capitalize: हर शब्द का पहला अक्षर बड़ा
    • iNVERT: बड़े को छोटा और छोटे को बड़ा
  3. रिजल्ट नीचे वाले बॉक्स में दिखाई देगा।
  4. Copy Output बटन से उसे कॉपी करें या Clear All से दोनों बॉक्स क्लियर करें।
